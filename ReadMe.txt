Phần mềm quản lý nhà thuốc tây

-source toàn bộ chương trình nguồn của đồ án.
-release chứa file thi hành EXE và các file thư viện(Lib, DLL, ...) 
-references chứa các file tài liệu tham khảo (DOC, PDF, HTML, source code, ...) 
mà đồ án có tham khảo
-chương trình phải được thử nghiệm trên môi trường HĐH windows
-Tạo bộ setup và kiểm tra chương trình trên các máy khác tránh tình trạng cài đặt 
thiếu thư viện, component, Active X.

Nhóm đại diện nộp file báo cáo trên Classroom (file .rar) bao gồm: CSDL,
Coding, nội dung trình bày báo cáo (file word, file PDF, file PowerPoint).
Trong buổi báo cáo:
- Nộp 1 quyển báo cáo.
- Trình bày báo cáo trên PowerPoint
- Demo chương trình

Các sơ đồ cần thiết kế BPM, ERD, BFD:
- BFD giao thức phát hiện lỗi
- ERD mô hình thực thể kết hợp
- BPM quản lý kinh doanh
